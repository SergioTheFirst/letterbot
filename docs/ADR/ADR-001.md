# ADR-001: Surprise Budget (shadow-only)

## Status
Accepted (shadow-only)

## Context
Нужно измерять неожиданные приоритетные коррекции без изменения поведения.

## Decision
- Фиксировать SURPRISE_DETECTED при ручной коррекции приоритета.
- Только запись события, без автокалибровок.

## Consequences
- Метрика Surprise Rate доступна из events_v1.
- Нет влияния на доставку до появления данных.
