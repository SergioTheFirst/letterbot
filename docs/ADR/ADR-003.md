# ADR-003: Silence as Signal (shadow)

## Status
Accepted (shadow-only)

## Context
Молчание пользователя может быть сигналом перегрузки.

## Decision
- Регистрировать SILENCE_SIGNAL_DETECTED на основе имеющихся событий.
- Без изменения доставки в v1.5.

## Consequences
- Появится источник данных для будущих моделей.
