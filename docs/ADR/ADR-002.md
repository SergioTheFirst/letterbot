# ADR-002: Circadian Delivery

## Status
Rejected (superseded by Constitution: immediate delivery only)

## Context
Constitution now forbids любые окна бездействия, batching, weekend rules, или отложенные уведомления.

## Decision
- Единственный режим доставки — немедленный Telegram Tier-1 для каждого письма.
- Дальнейшее обогащение только через edit-in-place того же сообщения.

## Consequences
- Никаких ночных/выходных исключений, батчей или тихих режимов.
- Поведение детерминировано: одинаковые входы → одинаковое немедленное решение.
