# ADR-002: Circadian Delivery

## Status
Accepted

## Context
Пользователь не должен получать некритичные уведомления ночью.

## Decision
- Вводится quiet window на основе локального времени машины.
- Некритичные письма ночью → DEFER_TO_MORNING или BATCH_TODAY.

## Consequences
- Fast First сохраняется для критичных писем.
- Решения фиксируются в DELIVERY_POLICY_APPLIED.
