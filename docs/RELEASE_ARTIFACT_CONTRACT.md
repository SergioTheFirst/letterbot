# Release Artifact Contract (Windows one-folder ZIP)

Product artifact: contents of `dist/MailBot/` packed as one-folder ZIP.

## Required files in `dist/MailBot/`
- `MailBot.exe`
- `run.bat`
- `config.example.yaml`
- `README_QUICKSTART_WINDOWS.md`
- `manifest.sha256.json`

## User-editable file
- Edit only `config.yaml` (created from `config.example.yaml` on first run).

## Must not be edited
- `MailBot.exe`
- `manifest.sha256.json`
- Internal packaged modules/files generated by PyInstaller.

## Expected first-run behavior
- If `config.yaml` is missing, `run.bat` creates it from `config.example.yaml`.
- Launcher prints a clear configuration-required message and opens `config.yaml` in Notepad.
- After saving `config.yaml`, user runs `run.bat` again.
