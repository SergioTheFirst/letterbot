# Release Artifact Contract (Windows one-folder ZIP)

Product artifact: contents of `dist/Letterbot/` packed as one-folder ZIP.

## Required files in `dist/Letterbot/`
- `Letterbot.exe`
- `run.bat`
- `mailbot_v26/config/settings.ini.example`
- `mailbot_v26/config/accounts.ini.example`
- `README_QUICKSTART_WINDOWS.md`
- `manifest.sha256.json`
- `UPGRADE.md`
- `SMARTSCREEN.md`
- `CHANGELOG.md`

## User-editable files
- `mailbot_v26/config/settings.ini`
- `mailbot_v26/config/accounts.ini`

These are created by `run.bat` from `*.example` on first run.

## Must not be edited
- `Letterbot.exe`
- `manifest.sha256.json`
- Internal packaged modules/files generated by PyInstaller.

## First run behavior
- If `settings.ini` or `accounts.ini` is missing, `run.bat` creates missing file(s) from examples.
- Launcher runs `Letterbot.exe config-ready --config-dir mailbot_v26\config --verbose`.
- If required account fields are missing, launcher opens `accounts.ini` in Notepad and retries (max 20 attempts).
- Required fields: `login, password, host, port, use_ssl`.

## Verification command
- `python -m mailbot_v26.tools.verify_dist dist/Letterbot`
